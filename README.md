# Arch Linux Dotfiles

My personal Arch Linux configuration and desktop setup for Arch Linux.

## Repository Structure

```text
.
├── .config/
│   ├── fastfetch/
│   ├── kitty/
│   ├── zed/
│   ├── easyeffects/
│   └── pavucontrol.ini
│
├── autostart/
├── easyeffects.backup/
│   └── db/
│
├── .p10k.zsh
├── .zpreztorc
├── .zprofile
├── .zshenv
├── .zshrc
└── README.md
```

## Included Configurations

* **Zsh** – Shell configuration, aliases, and environment setup
* **Powerlevel10k** – Terminal prompt customization
* **Prezto** – Zsh framework configuration
* **Kitty** – Terminal emulator configuration
* **Neovim** – Text editor setup (if included in your config)
* **Zed** – Modern code editor configuration
* **Fastfetch** – System information display
* **EasyEffects** – Audio processing profiles and settings
* **PavuControl** – Audio control configuration
* **Autostart** – Applications that run automatically on login

## Notes

This repository contains my personal Arch Linux setup. It is optimized for my workflow and may change frequently as I tweak my system.

Feel free to use it as a reference or adapt it for your own setup.
